# Apeke Off Bot 🚀

A multifunctional Telegram bot with image generation, conversion, URL shortening, and more!

## 🌟 Features

- 🎨 **AI Image Generation** - Create images from text prompts using Stable Diffusion
- 🔄 **Image Format Conversion** - Convert between PNG, JPG, WEBP, BMP, GIF, TIFF
- 🔗 **URL Shortening** - Shorten long URLs instantly
- 📏 **Image Resizing** - Resize images to any dimensions
- 🗜️ **Image Compression** - Reduce image file sizes with quality control
- 💬 **Echo Function** - Simple message test

## 🛠️ Tech Stack

- Python 3.9+
- python-telegram-bot v20.3
- Pillow (image processing)
- Hugging Face API (AI image generation)
- Railway (hosting)
- GitHub (version control)
